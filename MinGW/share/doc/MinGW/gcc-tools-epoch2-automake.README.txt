mingw gcc-tools-epoch2-automake
================================================================================
Provides a version of automake suitable for use by gcc developers. Gcc requires
an unpatched version of automake.  This package provides that software,
installed into the /mingw/opt/gcc-tools/epoch2 prefix.

epoch2 autotools are: autoconf-2.64 + automake-1.11.latest, and are
                      used for gcc-4.5.

Build instructions:
  unpack gcc-tools-epoch2-automake-1.11.1-X-mingw32-src.tar.lzma
  mgwport gcc-tools-epoch2-automake-1.11.1-X.mgwport all

This will create:
  gcc-tools-epoch2-automake-1.11.1-1-mingw32-src.tar.lzma
  gcc-tools-epoch2-automake-1.11.1-1-mingw32-bin.tar.lzza
  gcc-tools-epoch2-automake-1.11.1-1-mingw32-lic.tar.lzma
  gcc-tools-epoch2-automake-1.11.1-1-mingw32-doc.tar.lzma

See `mgwport --help' for other options.
